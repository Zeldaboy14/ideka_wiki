[OoT] Making items usable by adult/young Link

Have you ever wanted to use the hookshot as young Link in OoT? Probably not, but here is a tutorial on how to make adult Link-only  items usable by young Link and vice versa. For this example, let's make the "Megaton Hammer" usable by young Link.

You'll probably want to use this page (thanks, DeathBasket) to find the offsets for the item your going to change, unless you want to locate them yourself. As you can see, the "Offsets for item usability" offset for the Megaton Hammer is 0xBF9B1B. Open your debug ROM in a hex editor(just use HxD, please) and hit CTRL + G, and write that offset in the window that pops up.  This will take you to a byte reading "00". You can either change this to "01" (usable by young Link only) or "09" (usable by both). Now you will need to find the "Offsets for item highlight" offset for the Megaton Hammer, so scroll down to that section and find it. As you hopefully see, it is 0xBF9B45. Hit CTRL + G again and write that offset. This will take you to another byte Reading "00". Again change this to either 01, or 09, depending on if you want it only to be usable by young Link, or both of them. Now just save and test if you did it the right way.
