[MM] Importing a custom map

Note that this tutorial just covers how to simply load a single custom map in MM, nothing more. There may also be some issues with the map by only using this method.
What you’ll need:
  • Hex editor (HxD will be used in this tutorial)

  • .ZSCENE to replace (01F70000 - 01F77880.ZSCENE (the Last boss room) will be used in this tutorial)

  • Sharp Ocarina

  • Custom Map as an .OBJ file

  • Decompressed MM ROM (the american 1.0 version will be used in this tutorial)
Open up Sharp Ocarina and click File>New Scene. Now enter your wanted .OBJ file as the collision model. Switch over to the “Rooms” tab and, like you previously did with the collision model, enter the .OBJ file you want as the room. Now File>Save Binary and save it wherever you want.

Now open all files in HxD (the Last Boss .ZSCENE, the ROM, your custom .ZSCENE and your custom .ZMAP).  Firstly, locate the 0x03 command in the Last Boss scene, which should be near the beginning of the file and is formatted like this:

03 00 00 00 BB XX XX XX

BB = Bank
XX = Offset of the Collision header

Look at the XX bytes and go to the offset  that these points to.  You will arrive at the scene’s collision header, which is formatted like this:

]BBBBCCCC DDDDEEEE FFFFGGGG VVVV0000
AAAAAAAA PPPP0000 LLLLLLLL TTTTTTTT
XXXXXXXX WWWW0000 MMMMMMMM

You won't have to worry about most of this, just copy the X bytes.

Switch over to your custom scene file. By doing like you just did in the Last Boss scene, locate the file’s collision header. Replace its T bytes with the ones you copied from the previous file. Now null out the eventual M bytes with 00 padding. Next go to the absolute end of the file and paste these bytes there:

02 BC FF FF FF FD FF FF 00 01 00 FF 00 00 00 1B

02 58 FF FF FF FE FF FF 00 02 00 FF 00 00 00 1B 

02 BC FF FF FF FC FF FF 00 03 00 FF 00 00 01 1B 

02 BC FF FF FF FB FF FF 00 04 00 FF 00 00 00 1B 

01 F4 FF FF FF F9 FF FF 00 05 00 FF 00 00 00 20 

01 90 FF FF FF F5 FF FF 00 06 00 FF 00 00 01 20 

00 64 FF FF FF F8 FF FF 00 07 00 FF 00 00 00 20 

00 C8 FF FF FF F7 FF FF 00 08 00 FF 00 00 00 20 

03 20 FF FF FF FA FF FF 00 09 00 FF 00 00 00 20 

03 84 FF FF FF F0 FF FF FF FF 00 FF 00 00 01 20

Go back to the start of your scene and locate the 0x0E command. Replace this with a 0x1B command, which basically is formatted like this:

1B 0A 00 00 BB XX XX XX

As the XX bytes, write the offset of where you just pasted the bytes (the end of your file). As the bank, write 02.

Now open your custom .ZMAP file and copy all of it. Switch over to your decompressed MM ROM. Depending on the file size of your custom map file, there are different locations in ROM where you can paste it, since it doesn't seem to be possible to load files from the unused space in the ROM. If you're file is smaller than 0x8A60 bytes, I would recommend pasting it at offset 0x01F78000 in the ROM. This would overwrite the Last Boss room, which won't be used anyway if you replacing that scene like me. If it's bigger however, I would recommend you to paste it at offset 0x025C5000 in the ROM. Then you probably won't have to worry about space anymore, since this would overwrite Termina Field. Write down the start- and end offset of where your "newly-pasted" .ZMAP file is located at.

Switch back to your custom .ZSCENE file and locate the 0x04 command in the header. The last 0x3 bytes of the command (it is 0x8 bytes long) points to an offset in the scene. Go to this offset and paste the offsets you wrote down about your .ZMAP's location (which should be 0x8 bytes long). This will rewrite the pointers that decides from where in the ROM your map will be loaded. Now copy your whole scene file and go back to your ROM.

Again, depending of the size of your custom .ZSCENE, you should paste it at different locations. If it's smaller than 0x7880 bytes, You can paste it at offset 0x017F0000 in the ROM. If it's bigger, paste it at offset 0x0258C000 (this will overwrite the Termina Field scene file). Like you did with the .ZMAP, write down the start and end offset for where you pasted your .ZSCENE.

Go to offset 0x0 in the ROM, and search for "01F7000001F77880". Once you've done that, hit "F3" to find again. Here, paste the start- and end offset for your scene file that you just wrote down. Now save and test! Use this code to quickly load the Last Boss room:
]D03E6B3A 0000
E89346EB 5959
E99346EE 5965
