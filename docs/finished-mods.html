MM: The Golden Knight's Mask
I was looking around in some old folders in my computer when I found the old OBJ file I used for the character in the mod above, and thought that I might as well try to import it to MM, since I was (and am) not very familiar with modifying the game. At first I replaced Zora Link's model, but then I thought that it would fit better to replace Fierce Deity Link instead. This is also one of the first "play as" mods for MM (the third, actually).
